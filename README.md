# GoSwag-Books
Golang backend with Postgres DB

# Prerequisites
Install Go and add it to the PATH.

# Steps to get started
1. `go mod init GoSwag-Books`
2. `go mod tidy`
3. `go run main.go`

# To Access API Endpoints
1. http://localhost:8000/swagger/index.html
